What is it?

Advanced budget analysis, using actual financial data imported from GnuCash, Quicken, mint.com, etc. Written in Java, it runs on Linux, Windows, and OS X. Available as a command-line application and desktop graphical application. A mobile Android version is planned for future release.

Dependencies

Java 6

How to Install

Extract the download archive (zip or tar file) and run the jar file (for Windows and OS X, double-click the jar file, for Linux, run "java -jar underbudget.jar"
