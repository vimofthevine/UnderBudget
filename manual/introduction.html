Introduction to UnderBudget

UnderBudget is an application for creating and analyzing budgets by comparing against actual data from a separate ledger/checkbook application (GnuCash, Quicken, etc.).

Creating a Budget

Create a budget by setting up estimates or expense categories with estimated, or expected, expense or income values.

Importing Data

Import transactions GnuCash XML files, QIF, or OFX files.

Analyzing the Budget

Compare actual expenses to expected values.

Exporting Data

Export reports to HTML or PDF.
