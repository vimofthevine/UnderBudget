Installation

UnderBudget is written in Java, and is therefore cross-platform compatible with Linux, OS X, and Windows.

Note: At this time, no installers or custom launchers are available. All distributions are a zip file containing all necessary files and a platform-specific launch script.

To install, download to appropriate package and extract (WinZIP, tar, etc.) to the desired location on the target computer.

To run UnderBudget, use the launch script included in the extracted folder (underbudget.bat for Windows and underbudget.sh for Linux/Mac).

Note: Optionally, Linux users can run “make install” to copy files into the appropriate system folders. A package is available in the AUR for Arch users.
