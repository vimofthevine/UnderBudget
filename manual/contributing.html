Contributing to UnderBudget

UnderBudget is released under the MIT license and contributions are welcome!

Source is hosted on github and Vincent Driessen’s workflow for git is used for all development.

Reporting Issues

Please report any issues or errors encountered while using UnderBudget!

Issues are managed using the github issue tracker and may be filed directly on github by users with github accounts.

For all others, issues may be reported by creating a thread in the issues forum.

Building From Source

To build UnderBudget from source, clone the repository or download the source package from github. An ant script is provided to perform the Java compilation and packaging. Run “ant -projecthelp” from the project folder to see all available targets.
